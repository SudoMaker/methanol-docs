# 页面专用脚本与样式

Methanol 支持自动加载与页面同名且同目录的脚本及样式文件。

## 机制详解

示例结构：

```
pages/
  guide/
    components.mdx
    components.css
    components.js
```

访问 `/guide/components` 时，系统将自动把 `components.css` 与 `components.js` 注入至页面 `<head>` 中。

## 支持的文件扩展名

样式：

- `.css`

脚本：

- `.js`
- `.mjs`
- `.cjs`
- `.ts`
- `.mts`
- `.cts`

## 示例

```
pages/
  about.mdx
  about.css
  about.ts
```

`about.css` 与 `about.ts` 将仅在 `/about` 页面加载。

## 注意事项

- 文件路径解析以页面所在目录为基准。
- 全局资源（`pages/style.css`、`pages/index.js/ts`）依然对所有页面生效。
