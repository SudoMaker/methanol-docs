# 写作基础

Methanol 原生支持 Markdown 与 [MDX](https://mdxjs.com)。标准 Markdown 语法足以应对大多数文档结构与内容需求；若需更丰富的版式或交互体验，可引入 JSX 组件。

## 页面结构

```mdx
---
title: 我的页面
excerpt: 用于 meta 的简短摘要
---

# 我的页面

在这里撰写内容。
```

顶部以 `---` 包裹的内容即为 Frontmatter，用于定义页面配置与元数据（Metadata），详见下文。

## 标题与目录结构

`##`、`###`、`####` 将用于生成目录。层级越清晰，侧边栏目录的可读性越强。

## 正文段落

在 Markdown 中，正文通常由若干“段落”构成：

- 段落之间需用**空行**分隔。
- 同一段落内的换行通常会被视为空格（即不会触发强制换行）。
- 若确需“强制换行”，可在行尾添加两个空格，或直接写入 `<br />`（MDX 支持 JSX/HTML）。

示例：

```md
第一段第一行
第一段第二行（通常会和上一行合并）

第二段
这一行末尾有两个空格  
所以会换行
```

## 引用块

```mdx
> 这是一句引用内容
>
> 换行需要提供一个间隔，
> 连续的多行引用会被合并成一行

> 不连续的换行会被渲染为不同的引用块
```

渲染效果如下：

> 这是一句引用内容
>
> 换行需要提供一个间隔，
> 连续的多行引用会被合并成一行

> 不连续的换行会被渲染为不同的引用块

## 代码块

请使用带语言标识的围栏代码块（Fenced Code Block）：

<pre>
	<code class="language-md">
		{`\`\`\`js
export const title = 'Hello'
\`\`\``}
	</code>
</pre>

渲染效果如下：

```js
export const title = 'Hello'
```

## Frontmatter 配置要点

常用字段：

- `title`: 页面标题及导航显示名称
- `excerpt`: 用于 Meta 标签的简短描述
- `date`: 发布日期（ISO 格式）
- `weight`: 导航排序权重

完整字段说明请参阅 [Frontmatter 参考](../reference/frontmatter)。
