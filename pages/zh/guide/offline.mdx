# 离线访问 (PWA)



Methanol 内置了基于 Workbox 的 PWA 支持。其 Service Worker 实现经过高度优化，在确保内容实时更新的同时，能有效防止大型站点因资源过多而导致的安装停滞。



## 启用 PWA



在 `methanol.config.js` 配置文件中开启：



```js

export default () => ({

	pwa: true

})

```



### 自定义 Manifest



你可以根据需要自定义 PWA Manifest 及 `injectManifest` 配置：



```js

export default () => ({

	pwa: {

		manifest: {

			name: 'My Site',

			short_name: 'My Site'

		},

		injectManifest: {

			globPatterns: ['**/*.{js,css,html,ico,png,svg}']

		}

	}

})

```



### 进阶配置



若需完全控制 PWA 行为（如自定义 Service Worker 策略等），可设置 `pwa: false` 并在 `vite.plugins` 中自行配置 `vite-plugin-pwa`。



## 自定义离线页面



创建 `pages/offline.mdx` 即可自定义用户离线且请求资源未缓存时的回退页面。
