# 链接与导航

Methanol 使用基于文件的路由。站内链接直接写“路由路径”即可。

## 内部链接

使用站内路由（不带扩展名）：

```md
[写作基础](./writing)
```

渲染为：[写作基础](./writing)

页面移动或改名后，记得同步更新链接路径。

### 子路径部署（`site.base`）

如果站点部署在子路径下（例如 `https://example.com/docs/`），请设置 `site.base: '/docs/'`，并避免在内容里写 `/blog/post` 这类根路径的站内链接。

推荐使用相对链接：

```md
[下一篇](./next-post)
[返回博客](../blog/)
```

如果在 MDX/JSX 中需要生成根路径链接，请用 `ctx.withBase` 包一下：

```mdx
<a href={ctx.withBase('/blog/post')}>阅读</a>
```

## 外部链接

外部站点使用完整 URL：

```md
[Example](https://example.com)
```

渲染为：[Example](https://example.com)

## 导航排序

通过 Frontmatter 控制顺序与可见性：

- `weight`: 数值越小越靠前
- `hidden`: 在侧边栏隐藏
- `isRoot`: 用于 `index.mdx`，定义导航根

更完整的规则见 [Frontmatter 参考](../reference/frontmatter)。
