# 链接与导航

Methanol 使用基于文件的路由。站内链接直接写“路由路径”即可。

## 内部链接

使用站内路由（不带扩展名）：

```md
[写作基础](/zh/guide/writing)
```

渲染为：[写作基础](/zh/guide/writing)

页面移动或改名后，记得同步更新链接路径。

## 外部链接

外部站点使用完整 URL：

```md
[Example](https://example.com)
```

渲染为：[Example](https://example.com)

## 导航排序

通过 Frontmatter 控制顺序与可见性：

- `weight`: 数值越小越靠前
- `hidden`: 在侧边栏隐藏
- `isRoot`: 用于 `index.mdx`，定义导航根

更完整的规则见 [Frontmatter 参考](/zh/reference/frontmatter)。
