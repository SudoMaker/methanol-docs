# GitHub 风格 Markdown 扩展（GFM）

本页面旨在演示 Methanol 环境下 Markdown/MDX 对 GitHub Flavored Markdown (GFM) 特性的支持情况。

在调试 Markdown 处理链路（解析、渲染、主题样式）时，本页可作为标准的“对照样本”。

## GFM 简介

GFM 是由 GitHub 维护的 Markdown 方言（扩展规范），在标准 Markdown 基础上引入了更契合技术写作与协作的语法特性，包括：

- 表格
- 任务列表（Checkbox）
- 删除线
- 脚注
- 自动链接（URL 或邮箱地址将自动转化为链接）

在 Methanol 中，GFM 默认处于开启状态。若需关闭，请在配置中进行如下设置：

```js
// methanol.config.js
export default () => ({
	gfm: false
})
```

参阅：[配置参考](./configuration)
