# GitHub 风格 Markdown 扩展（GFM）

这个页面用来快速演示 Methanol 的 Markdown/MDX 中常见的 GitHub Flavored Markdown（GFM）特性。

当你在调试 Markdown 处理链路（解析、渲染、主题样式）时，本页可以作为“对照样本”。

## 什么是 GFM？

GFM 是 GitHub 使用的 Markdown 方言（扩展规范）。它在基础 Markdown 之上增加了一些更贴近写作与协作的语法，例如：

- 表格
- 任务列表（复选框）
- 删除线
- 脚注
- 自动链接（直接写 URL/邮箱也会被识别为链接）

在 Methanol 里，GFM 默认开启。如果你需要关闭它，可以在配置里设置：

```js
// methanol.config.js
export default () => ({
	gfm: false
})
```

另见：[配置（Configuration）](./configuration)

当你想验证 Markdown 处理链路（MDX/MD 解析、样式、主题渲染）时，这页可以作为“对照样本”。

## 脚注（Footnote）

A note[^1]

[^1]: Big note.

## 删除线（Strikethrough）

~one~ or ~~two~~ tildes.

## 表格（Table）

| - | :- | -: | :-: |
| 1 | 2 | 3 | 4 |

## 任务列表（Tasklist）

* [ ] to do
* [x] done
