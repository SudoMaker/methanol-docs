# 欢迎使用 Methanol！

	<LinkButton variant="primary" href="./quick-start">快速上手</LinkButton>

Methanol 是一个基于 [rEFui](https://github.com/SudoMaker/rEFui) 与 [MDX](https://mdxjs.com) 的静态站点生成器。开箱即用，同时也支持深度定制；启动迅速、结构清晰，旨在让你专注于内容创作。

几乎没有配置门槛：将文件置入 `pages/`（或 `docs/`），路由即刻生成；`index.mdx` 默认作为首页，导航体系亦会自动构建。

## 核心特性

- 将 MD/MDX 文件放入 `pages/`（或 `docs/`）目录，即可自动生成路由与导航
- 在 `components/` 中编写可复用组件，直接嵌入 MDX 文档中使用
- 利用 Frontmatter 精确控制导航结构、排序与可见性
- 构建输出至 `dist/`，支持本地预览与发布
- 可选集成 [Pagefind](https://pagefind.app) 搜索与 [Starry Night](https://github.com/wooorm/starry-night) 语法高亮

## rEFui：HTML + Islands

Methanol 在服务端 HTML 生成及客户端交互层面均采用 rEFui。页面主体以静态 HTML 形式输出，仅需交互的客户端组件将作为微型 “islands” 进行局部注水（Hydration），避免全页重绘。
