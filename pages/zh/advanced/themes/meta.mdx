# 默认主题元标签

默认主题仅在存在有效值时渲染元标签（或回退至默认设定）。

## Logo 与 Favicon 解析优先级

优先级顺序如下：

1) 当前页面 `frontmatter.logo` / `frontmatter.favicon`
2) 根目录索引页面的 `frontmatter.logo` / `frontmatter.favicon`
3) 全局配置 `site.logo` / `site.favicon`
4) 主题默认预设（`/logo.png`, `/favicon.png`）

若需显式禁用 Logo/Favicon，可将 `logo` 或 `favicon` 设为 `false`（支持 Frontmatter 或 Config 配置）。设置为 `false` 将终止解析并跳过所有回退逻辑（包括主题默认值）。

若最终解析结果为 Falsy 值，对应的 `<link rel="icon">`（或 Logo `<img>`）将不会被渲染。

## 页面描述（Meta Description）

- `frontmatter.excerpt`（别名：`description`）将被渲染为 `<meta name="description" content={...} />`。

## Open Graph (OG) 协议

- `frontmatter.ogTitle` → `og:title`
- `frontmatter.ogDescription` → `og:description`
- `frontmatter.ogImage` → `og:image`
- `frontmatter.ogUrl` → `og:url`

## Twitter Card

- `frontmatter.twitterCard` → `twitter:card`
- `frontmatter.twitterTitle` → `twitter:title`
- `frontmatter.twitterDescription` → `twitter:description`
- `frontmatter.twitterImage` → `twitter:image`
