# オフライン対応 (PWA)

Methanol は Workbox をベースとした PWA 機能を標準でサポートしています。**高度に最適化された** Service Worker 実装を採用しており、頻繁なコンテンツ更新に柔軟に対応しつつ、大規模なサイトでもインストールの停滞を招くことなくスムーズに動作するよう設計されています。

## PWA の有効化

`methanol.config.js` において、以下の設定で有効化できます。

```js
export default () => ({
	pwa: true
})
```

### Manifest のカスタマイズ

PWA の manifest と injectManifest 設定を上書きできます。

```js
export default () => ({
	pwa: {
		manifest: {
			name: 'My Site',
			short_name: 'My Site'
		},
		injectManifest: {
			globPatterns: ['**/*.{js,css,html,ico,png,svg}']
		}
	}
})
```

### さらに細かく制御する場合

PWA を完全にカスタマイズしたい場合は `pwa: false` を設定し、
`vite.plugins` で自分の `vite-plugin-pwa` 設定を追加してください。

## オフラインページ

`pages/offline.mdx` を作成することで、オフライン用のフォールバックページをカスタマイズできます。
