# リンクとナビゲーション

Methanol はファイルベースルーティングを採用しています。内部リンクはルートパスで指定します。

## 内部リンク

拡張子なしのルートでリンクします。

```md
[執筆の基本](/ja/guide/writing)
```

表示例: [執筆の基本](/ja/guide/writing)

ページを移動・リネームしたらリンク先も更新してください。

## 外部リンク

外部サイトはフル URL を使います。

```md
[例](https://example.com)
```

表示例: [例](https://example.com)

## ナビゲーション順

フロントマターで順序や表示/非表示を制御できます。

- `weight`: 数値が小さいほど上に並ぶ
- `hidden`: サイドバーから非表示
- `isRoot`: `index.mdx` に付けてナビゲーションのルートを指定

詳しくは [Frontmatter リファレンス](/ja/reference/frontmatter) を参照してください。
