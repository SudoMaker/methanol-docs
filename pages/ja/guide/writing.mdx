# 執筆の基本

Methanol は Markdown と [MDX](https://mdxjs.com) をサポートします。構成は Markdown で書き、よりリッチな表現が必要なときに JSX コンポーネントを使えます。

## 基本ページ

```mdx
---
title: マイページ
excerpt: meta タグ用の短い説明
---

# マイページ

ここに本文を書きます。
```

`---` で囲まれたブロックはフロントマター（frontmatter）です。役割はこの後で説明します。

## 見出しと目次

`##`、`###`、`####` の見出しが目次に使われます。階層を整えるとサイドバーの目次が読みやすくなります。

## 本文

Markdown の本文は基本的に「段落」で構成されます:

- 段落は**空行**で区切ります。
- 同じ段落内の改行は、通常は空白として扱われます（強制改行にはなりません）。
- 強制改行したい場合は行末に半角スペースを 2 つ入れるか、`<br />` を使います（MDX は JSX/HTML を書けます）。

```md
第1段落 1行目
第1段落 2行目（通常は同じ段落として扱われます）

第2段落
この行の末尾にはスペースが2つあります  
なので改行されます
```

## 引用

```mdx
> これは引用（Blockquote）です。
>
> `>` だけの行を入れると、同じ引用ブロック内で空行を表現できます。
> 複数行の引用は 1 つの引用ブロックになります。

> `>` のブロック間に空行があると、別の引用ブロックとして扱われます。
```

次のように表示されます:

> これは引用（Blockquote）です。
>
> `>` だけの行を入れると、同じ引用ブロック内で空行を表現できます。
> 複数行の引用は 1 つの引用ブロックになります。

> `>` のブロック間に空行があると、別の引用ブロックとして扱われます。

## コードブロック

言語名付きのフェンスを使います。

<pre>
	<code class="language-md">
		{`\`\`\`js
export const title = 'Hello'
\`\`\``}
	</code>
</pre>

次のように表示されます:

```js
export const title = 'Hello'
```

## フロントマターのヒント

よく使うフィールド:

- `title`: ページタイトルとナビゲーション名
- `excerpt`: メタタグ用の短い説明
- `date`: 公開日（ISO 形式）
- `weight`: ナビゲーションの順序

一覧は [Frontmatter リファレンス](/ja/reference/frontmatter) を参照してください。
