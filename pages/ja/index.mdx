# Methanol へようこそ！

<ButtonGroup>
	<LinkButton variant="primary" href="./quick-start">クイックスタート</LinkButton>
	<LinkButton variant="secondary" href="https://github.com/SudoMaker/Methanol" target="_blank">GitHub</LinkButton>
</ButtonGroup>

Methanol は、[rEFui](https://github.com/SudoMaker/rEFui) と [MDX](https://mdxjs.com) を基盤とした、柔軟なカスタマイズ性と使いやすさを両立した静的サイトジェネレーターです。高速な動作と洗練された構造により、執筆に集中できる環境を提供します。

複雑な設定は必要ありません。`pages/`（または `docs/`）にファイルを配置するだけでルートが自動生成され、`index.mdx` はホームページとして、ナビゲーションツリーも自動的に構築されます。

## 主な特徴

- `pages/`（または `docs/`）への MD/MDX 配置による自動ルーティング
- `components/` に作成したコンポーネントを MDX 内で直接利用可能
- フロントマターによるナビゲーション構造、順序、表示のきめ細かな制御
- `dist/` へのビルド出力とローカルでのプレビュー確認
- [Pagefind](https://pagefind.app) による全文検索や [Starry Night](https://github.com/wooorm/starry-night) による構文ハイライトの統合

## rEFui: HTML + Islands

Methanol は、サーバーサイド HTML 生成とクライアントサイドのインタラクションの両方に rEFui を採用しています。ページの大半は静的な HTML として出力され、インタラクティブな要素が必要な箇所のみが小さな “islands” として部分的にハイドレーション（Partial Hydration）されます。これにより、全ページをハイドレーションすることなく、高いパフォーマンスを実現しています。
