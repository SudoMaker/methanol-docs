# ページインデックス（`methanol:pages`）

Methanol は、ブログ記事の一覧表示、カテゴリーによるフィルタリング、カスタム検索 UI などのクライアント側機能のために、全ページの一覧を提供する仮想モジュールを用意しています。

```js
import pages from 'methanol:pages'
// または
import { pages } from 'methanol:pages'
```

## 含まれる内容

インデックスの各エントリにはページのメタデータが含まれますが、インデックスを軽量に保つために、データ量の大きいフィールドは除外されています。

- **除外されるフィールド:** `content`、`mdxComponent`、`mdxCtx`、`getSiblings`
- **除外される値:** 関数型のフィールドすべて
- **常に含まれるフィールド:** `excerpt`。フロントマターで指定されていない場合、Methanol は本文から自動生成します。
- **除外対象:** `hidden: true` が設定されたページはインデックスに含まれません。

生成されるオブジェクトには、`routePath`、`routeHref`、`title`、`frontmatter`、`stats`、`excerpt` などの情報が含まれます。

## 備考

- このモジュールは、**クライアント側** でのロジック構築専用に設計されています。
- 開発モード（dev）では、ページが変更されるたびにインデックス内容も自動的に更新されます。
