# `Head` / `head`

Methanol は、すべての `.mdx` ページにおいて `Head` コンポーネント（別名: `head`）を標準で提供しています。明示的なインポートなしで直接利用可能です。

`Head` コンポーネント内に記述した要素は自動的に収集され、最終的な HTML の `<head>` セクションに挿入されます（テーマ側で `<ExtraHead />` を適切にレンダリングしている必要があります。デフォルトテーマはこれに対応済みです）。

## 使用例

```html
<Head>
	<meta name="description" content="ページの説明文" />
	<link rel="canonical" href="https://example.com/docs/page" />
</Head>
```

## 自動注入される静的アセット

Methanol は、以下のルールに基づいてアセットを自動的に `<head>` へ挿入します。

- `pages/style.css` (存在する場合)
- `pages/index.js` / `pages/index.ts` (存在する場合)
- 各 MDX ファイルと同名かつ同ディレクトリに配置された専用の CSS / JS / TS (例: `guide.mdx` の場合は `guide.css`, `guide.js`, `guide.ts` など)
