# CLI

Methanol includes a dedicated Command-line Interface (CLI) designed to streamline development, execution of static builds, and local previews.

## Core Commands

Quick start with `npx` (recommended):

```bash
npx methanol dev      # Initialize the development server
npx methanol build    # Execute production build to the dist/ directory
npx methanol serve    # Preview the production build locally
```

For improved performance and faster startup times, consider a global installation:

```bash
npm install methanol -g
methanol dev
```

The `input` and `output` paths can be provided as positional arguments (order is significant):

```bash
npx methanol build ./pages ./dist
```

In `dev` mode, only the `input` argument is utilized:

```bash
npx methanol dev ./pages
```

## Flag Reference

- `--input`, `-i`: Specifies the source directory (overrides the default `pages/`).
- `--components`: Specifies the components directory (overrides the default `components/`).
- `--assets`: Specifies the static assets directory (overrides the default `public/`).
- `--output`, `-o`: Specifies the build destination (overrides the default `dist/`).
- `--config`, `-c`: Defines the path to a custom configuration file.
- `--site-name`: Overrides the site title defined in configuration.
- `--base`: Temporarily overrides `site.base` for the current execution (ignored in `dev` mode).
- `--port`: Sets the port for the development and preview servers (default: 5173).
- `--host`: Binds the server to a specific host (using `--host` without a value binds to all interfaces).
- `--intermediate-dir`: Directory for intermediate HTML output (build only).
- `--emit-intermediate`: Enables the output of intermediate HTML to `build/` (build only).
- `--highlight`: Controls syntax highlighting (`--highlight` to enable; `--highlight false` to disable).
- `--search`: Controls Pagefind search indexing (`--search` to enable; `--search false` to disable).
- `--pwa`: Controls PWA support (`--pwa` to enable; `--pwa false` to disable).
- `-v`, `--verbose`: Enables detailed logging, including Vite and Pagefind output during the build process.

Syntax highlighting is disabled by default to optimize performance and can be enabled per-execution via the flag.

## Execution via Node

```bash
node node_modules/methanol/bin/methanol.js dev
```
