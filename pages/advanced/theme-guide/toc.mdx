# Rendering a Table of Contents (ToC)

Methanol automatically extracts headings during the page compilation process and provides them as structured data within `ctx.page.toc`.

Each entry in the ToC follows this structure:

```js
{
	depth: 2,
	id: 'my-heading',
	value: 'My heading',
	children: []
}
```

## Simple ToC Renderer Implementation

```jsx
const Toc = ({ toc }) => {
	if (!toc?.length) return null
	return (
		<nav aria-label="On this page">
			<div>On this page</div>
			<ul>
				{toc.map((item) => (
					<li>
						<a href={`#${item.id}`}>{item.value}</a>
						{item.children?.length ? <Toc toc={item.children} /> : null}
					</li>
				))}
			</ul>
		</nav>
	)
}

export default function PageTemplate({ PageContent, ExtraHead, ctx }) {
	return (
		<>
			<html>
				<head>
					<ExtraHead />
				</head>
				<body>
					<main>
						<PageContent />
					</main>
					<aside>
						<Toc toc={ctx.page.toc} />
					</aside>
				</body>
			</html>
		</>
	)
}
```
