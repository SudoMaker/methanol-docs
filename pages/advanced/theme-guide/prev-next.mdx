# Adjacent Page Links

Methanol provides a specialized helper for implementing "Previous" and "Next" page navigation:

```js
const { prev, next } = ctx.getSiblings() || {}
```

The resulting sequence is perfectly aligned with the current navigation root and the display order in the sidebar.

## Implementation Example

```jsx
const PrevNext = ({ ctx }) => {
	const siblings = ctx.getSiblings()
	const prev = siblings?.prev
	const next = siblings?.next
	if (!prev && !next) return null
	return (
		<nav aria-label="Page navigation">
			{prev ? <a href={prev.routeHref}>← {prev.title}</a> : <span />}
			{next ? <a href={next.routeHref}>{next.title} →</a> : <span />}
		</nav>
	)
}

export default function PageTemplate({ PageContent, ExtraHead, ctx }) {
	return (
		<>
			<html>
				<head>
					<ExtraHead />
				</head>
				<body>
					<main>
						<PageContent />
						<PrevNext ctx={ctx} />
					</main>
				</body>
			</html>
		</>
	)
}
```

## Best Practices

- **Use `routeHref`**: Ensure links point to `routeHref` to correctly account for `site.base`.
- **Filtering**: If you want to exclude specific pages from the adjacent navigation sequence, you can utilize `hidden: true` or similar frontmatter flags to remove them from the navigation tree.
